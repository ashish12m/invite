# Khushboo & Ashish — Wedding Website

An elegant destination wedding website for Khushboo & Ashish, celebrating at The Grand Bahubali (TGB), Balotra, Rajasthan.

**Date:** 30 April – 1 May, 2026

## Features

- Interactive envelope hero with "Click to Open" reveal
- Save the Date section
- Quick links (RSVP, Travel, Details, FAQs)
- The Finer Details & Dress Code
- Wedding Day Schedule
- Our Story
- FAQ section

## Tech

Single HTML file with embedded CSS and JavaScript. Fully responsive.

## Run locally

Open `index.html` in a browser, or use a simple server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000
